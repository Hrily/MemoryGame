How to play:

Memory Game is a simple puzzle type game in which player has to retype the string of numbers to win. There are two different modes in this game:
1. Classic:
In Classic mode, there are levels of strings. In each level, a string of numbers is flashed and the player has to memorize it and retype it exactly the same and in same exact sequence. The length of string increases with levels. Win 10 strings to go to next level. Player can play from level 1 or can choose any level he likes.
2. Series In Series mode, string lenght starts from 1 and gradually increases as the player retypes correctly. The string numbers are same but a new number is added everytime. The world record for this mode is about 99 numbers. How long you can survive?
